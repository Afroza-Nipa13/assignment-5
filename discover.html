Question-1: What are the different ways to select an element in the DOM?
The meaning of DOM is Document Object Model.There are so many ways to select elements.Most commonly used methods is : getElementById(): When you know the unique id of an element. 1.getElementsByClassName(): When selecting elements by class. 2.getElementsByTagName(): When selecting elements by their tag name. 3.querySelector(): To select an element with a classList. 4.querySelectorAll(): To select multiple elements to find a type of elements. 6.parentNode / childNodes: To navigate through the DOM tree from a given element.
